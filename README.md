# PROJECT GUILDELINES

## Prerequisites

* NodeJS **v18.10.0**
* @nestjs/cli **v10.1.18**
* Create a file named `.env `and fills in all values of variables defined in `.env.example`

### Installation

```
npm install -g @nestjs/cli@10.1.18

npm install
```

### Running the app

```
npm run start:dev
```
